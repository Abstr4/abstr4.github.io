About Me

How I got here

I started coding at university in 2018. My first languages were C++ and Haskell — learning algorithms, data structures, and problem solving. I also picked up Python for mathematical analysis and data visualization. I studied at Universidad Nacional de Córdoba (UNC) until 2021, working toward a B.S. in Computer Science.

After that, I took some time to figure things out. When I was ready to dive back in, I chose .NET as my focus in 2022 and haven't looked back since.

Building real things

My first production project was an inventory management system for a retail clothing store (TodoSeUsa). I started with ASP.NET Core MVC in December 2023, then rebuilt it with Blazor Server and Clean Architecture — that version launched in March 2025 and is still running today.

Alongside that, I built a suite of Discord bots for gaming and crypto communities (December 2024 - August 2025), handling real-time tracking, automated notifications, and data persistence with MySQL.

That project taught me something I still carry with me: good code isn't just code that works. It's code you can come back to six months later and actually understand what you wrote.

  • TodoSeUsa v1 (MVC): Dec 2023 — Mar 2025
  • TodoSeUsa v2 (Blazor): Dec 2023 — Present
  • Discord Bot Suite: Dec 2024 — Aug 2025

Where I'm at now

Since October 2025, I've been working as a C# .NET Fullstack Developer. Right now I'm deep in the .NET ecosystem, exploring how AI can speed up development without cutting corners on code quality.

I'm based in Córdoba, Argentina, and I'm always open to chatting about new projects or opportunities.
